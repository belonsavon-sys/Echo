# KPI and Metrics

## Product Health
- Daily active users (DAU)
- Monthly active users (MAU)
- Retention (D7, D30)

## Reliability
- Uptime
- API error rate
- Authentication failure rate

## Delivery
- Sprint completion rate
- Lead time from request to release
- Defect leakage rate

## Financial Signals
- Cost per active user
- Paid conversion rate (when monetization starts)
