# Product Strategy

## Product Vision
Build the most practical personal budgeting app for users who need clarity, not complexity.

## Product Principles
- Monthly-first budgeting model
- Fast entry and editing speed
- Explainable calculations
- Privacy by default
- Small, reliable feature set over broad feature sprawl

## Strategic Pillars
- Core budgeting excellence
- Reliable recurring behavior
- Professional but simple UX
- Account-level data isolation

## Success Criteria
- Users can set up and run a monthly budget in under 10 minutes
- Users trust totals and reports
- New requests can ship in short sprint cycles without regressions
